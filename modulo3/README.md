# Módulo 3 — Visualização de Dados com Matplotlib e Seaborn

Aprenda a criar gráficos informativos e visualmente atraentes para comunicar suas análises de dados. Este módulo cobre desde a anatomia básica de um gráfico até visualizações estatísticas avançadas, usando as bibliotecas Matplotlib e Seaborn.

## Conteúdo

- Anatomia de um gráfico (títulos, eixos, legendas)
- Gráficos básicos com Matplotlib (barras, linhas, pizza, histograma, boxplot)
- Visualizações estatísticas com Seaborn (boxplot, violin, heatmap, regressão)
- Customização de cores, estilos e temas
- Boas práticas de design de informação
- Escolha do gráfico ideal para cada tipo de dado
