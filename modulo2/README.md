# Módulo 2 — Manipulação de Dados com Pandas

Trabalha com dados reais do início ao fim: leitura, limpeza, manipulação e estatística descritiva usando Pandas e NumPy.

## Conteúdo

- Introdução ao Pandas (`Series`, `DataFrame`)
- Leitura e escrita de arquivos (CSV, JSON)
- Limpeza de dados (valores nulos, duplicatas, tipos)
- Seleção e manipulação (`loc`, `iloc`, filtros, `groupby`)
- Estatística descritiva com Pandas e NumPy
