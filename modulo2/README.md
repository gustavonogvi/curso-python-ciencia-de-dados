# Módulo 2 - Manipulação de Dados com Pandas

Esse é o segundo módulo do Curso de Python para Ciência de Dados. Aqui você aprenderá a usar a biblioteca Pandas para trabalhar com dados estruturados:

- Introdução ao DataFrame
- Leitura de dados (CSV, Excel)
- Limpeza de dados (valores nulos, duplicados)
- Manipulação de dados (filtros, ordenação, novas colunas, agrupamento)

## Instruções para executar o código

### No Google Colab

Você pode abrir os notebooks diretamente no Google Colab de duas formas:

- Abrir do GitHub:
	- Acesse https://colab.research.google.com/github/gustavonogvi/curso-python-ciencia-de-dados/blob/main/modulo2/modulo2_pandas.ipynb

- Fazer upload local:
	1. Acesse https://colab.research.google.com
	2. Clique em "Upload" e selecione o arquivo `modulo2_pandas.ipynb` ou `exercicios_modulo2.ipynb`.

- Crie uma cópia do notebook se não conseguir executá-lo.

### Na sua máquina

Primeiro, clone o repositório:

```bash
git clone https://github.com/gustavonogvi/curso-python-ciencia-de-dados.git
cd curso-python-ciencia-de-dados
```

Crie um ambiente virtual e ative-o:

```bash
python3 -m venv .venv
source .venv/bin/activate  # para Linux e MacOS
.\.venv\Scripts\activate  # para Windows
```

Instale as dependências:

```bash
pip install jupyter pandas
```

Inicie um servidor Jupyter:

```bash
jupyter notebook
```

Acesse o servidor em:

`http://localhost:8888`
