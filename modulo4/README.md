# Módulo 4 — Análise Exploratória de Dados (EDA)

Aprenda a investigar datasets reais de forma estruturada: levantando hipóteses, gerando visualizações e transformando achados em conclusões acionáveis.

## Conteúdo

- Carregamento e inspeção inicial de dados reais
- Formulação de perguntas de negócio
- Visualizações orientadas à análise (boxplot, correlação)
- Identificação e investigação de outliers
- Storytelling com dados: narrativa + gráfico + conclusão
- Síntese de insights em linguagem não técnica
