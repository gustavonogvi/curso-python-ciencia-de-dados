# Projeto Final — Análise de Qualidade de Vinhos

Trabalho de conclusão da disciplina. Aplica de forma integrada os conceitos dos quatro módulos do curso em um dataset real, respondendo perguntas de negócio com código, visualizações e interpretação dos resultados.

## Dataset

**Wine Quality** (UCI Machine Learning Repository) — 6.497 amostras de vinhos tintos e brancos portugueses, com 11 variáveis físico-químicas e uma nota de qualidade sensorial (0–10) atribuída por especialistas.

## Perguntas de negócio

1. O teor alcoólico influencia a qualidade percebida?
2. A acidez volátil prejudica a avaliação do vinho?
3. O açúcar residual é determinante para a qualidade?
4. Existem diferenças estruturais entre vinhos tintos e brancos?
5. Quais amostras apresentam comportamento atípico (outliers)?
